# Compilers
Compilers was a 3rd semester course where we had to develop:
- a deterministic pushdown automaton (DPA) using C+
- a grammar in Backus-Naur Form (BNF) using C++,
- a pushdown parser in C++,
- a lexical analyzer using Flex.
